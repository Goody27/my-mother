# User Stories — MyMom

## Epic 1: Slack Decline Agent (MVP)

### US-01: Auto Decline Unwanted Requests
**As** 山田くん (a new engineer who can't say no),  
**I want** MyMom to automatically decline Slack DMs requesting my time or attendance,  
**So that** I never have to feel guilty about saying no or spend mental energy on the right words.

**Acceptance Criteria:**
- [ ] MyMom detects new Slack DMs within 60 seconds
- [ ] Bedrock Guardrails passes before any message is sent
- [ ] A polite decline message is sent to the requester without user action
- [ ] User receives "断っておいたよ" notification only after completion
- [ ] User can cancel within 3 seconds of the "about to send" notification

### US-02: Cancel Before Send
**As** 山田くん,  
**I want** a 3-second window to cancel MyMom's action before it's sent,  
**So that** I feel in control and can intervene for important relationships.

**Acceptance Criteria:**
- [ ] Cancellation button appears immediately after MyMom decides to act
- [ ] Clicking cancel stops the send and confirms cancellation
- [ ] After 3 seconds with no cancellation, message is sent automatically

## Epic 2: Chat UI with Quick Replies

### US-03: Chat with Quick Reply Buttons
**As** 山田くん,  
**I want** MyMom to present 2 reply options after every message,  
**So that** I can continue the conversation with one tap and never have to think about what to say.

**Acceptance Criteria:**
- [ ] Every MyMom response includes exactly 2 quick-reply button options
- [ ] Tapping a button sends the selected text as the next message
- [ ] Free-text input is still available alongside buttons
- [ ] Buttons are contextually relevant (generated by Bedrock, not static)

---

## Epic 3: Personality Analysis & Sharing

### US-04: Personality Profile Card
**As** 山田くん (after 1 week of use),  
**I want** to see a personality card showing what MyMom has learned about me,  
**So that** I can share it with friends and feel understood.

**Acceptance Criteria:**
- [ ] Card generated after 1 week of usage (not 1 month)
- [ ] Shows 4 scores: 断り苦手度, 先延ばし傾向, 完璧主義度, 承認欲求度
- [ ] Shareable via Twitter Card format
- [ ] Sharing is opt-in; card is private by default

### US-05: Personalized AI Responses
**As** 山田くん,  
**I want** MyMom's responses to feel increasingly tailored to me over time,  
**So that** it feels like she actually knows me.

**Acceptance Criteria:**
- [ ] Personality profile is injected into Bedrock system prompt for all responses
- [ ] Profile updates weekly based on latest behavior data
- [ ] Users notice more relevant advice/tone as profile matures

---

## Epic 4: Responsibility SLA (Paid Plan)

### US-06: MyMom Fixes Her Own Mistakes
**As** 山田くん (paid plan),  
**I want** MyMom to automatically apologize and recover if her response caused a problem,  
**So that** I never have to deal with the fallout of her errors.

**Acceptance Criteria:**
- [ ] Failure detected automatically
- [ ] Apology message generated and sent to affected party
- [ ] Recovery actions executed
- [ ] User notified only after resolution: "対処しておいたよ"
